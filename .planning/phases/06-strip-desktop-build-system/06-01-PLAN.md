<objective>
Strip every desktop (PLATFORM_DESKTOP) path from the build system. Remove the
two desktop CMake branches (the `elseif (PLATFORM_DESKTOP)` link branch and the
`if (PLATFORM_DESKTOP)` packaging branch) and the dual-target header/inline
comments from `CMakeLists.txt`, then delete the two desktop helper scripts.

This satisfies BUILD-01 (no PLATFORM_DESKTOP in CMakeLists.txt) and BUILD-02
(desktop helper scripts removed). Phase 5 already collapsed the source seams to
Switch-only, so removing the build's desktop path leaves a tree that still builds.

Purpose: Make the build system offer only the Switch toolchain, cutting the
dual-target maintenance surface — with zero change to the shipped `.nro` (the
Switch link + packaging branches are preserved exactly).
Output: A Switch-only `CMakeLists.txt`; `scripts/build-desktop.sh` and
`scripts/run-desktop.sh` deleted.
</objective>
