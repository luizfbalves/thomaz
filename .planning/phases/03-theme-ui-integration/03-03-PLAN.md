## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| load_extract_state() → refreshExtractionStatus() | Persisted fw_version string used in UI label — never passed to privileged API |
| get_console_firmware() → fw_to_string() | IPC result used for display comparison only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-08 | Information Disclosure | mismatch advisory showing version strings | accept | Firmware version is not sensitive; advisory is display-only; both the recorded and current version are visible to the device owner |
| T-03-09 | Denial of Service | get_console_firmware() IPC call in refreshExtractionStatus() | accept | Single call on UI thread, not in a loop or frame callback; IPC latency is < 1ms each; acceptable per RESEARCH Pitfall 3 analysis |
| T-03-10 | Tampering | Stale status row after extraction | mitigate | refreshExtractionStatus() called inside brls::sync immediately after extraction completes; in-place label updates via stored raw pointer (Pitfall 1 prevention) |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No package manager installs in this plan |
</threat_model>
