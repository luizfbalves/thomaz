thomaz

Saves, cheats, themes, and games — all in one app

A friendly UI on your Switch, with free cloud save backup.

Download thomaz.nro View on GitHub

Features

🎮

Games

Lists installed games with icon, version, and active cheat badges.

🧩

Cheats

Downloads cheats from switch-cheats-db, toggles them on/off, saves to SD.

💾

Saves

Free cloud backup — save and restore your game saves with a thomaz account.

🎨

Themes

Browse and install community themes on your Switch.

All in one app Free cloud backup Bilingual PT-BR / EN Touch + controller

Requirements

  • Nintendo Switch with Atmosphère (CFW) and a homebrew menu (hbmenu)
  • Internet connection on the console
  • SD card space

Install

  1. Download thomaz.nro from Releases.
  2. Copy the file to the /switch/ folder on your SD card:
    sd:/switch/thomaz.nro
  3. On the Switch, open the Album (hold R to enter hbmenu) and launch thomaz.

Future updates can be done from the app via Settings → Check for updates.

How to use

  1. Open thomaz with your games closed.
  2. Tap the Cheats card → pick a game from the list.
  3. Enable the toggles you want — changes save automatically to SD.
  4. Close thomaz and launch the game. Atmosphère applies cheats on boot.

To remove cheats: home → Cheats → Clear cheats → select games → Clear selected.

Cheats are written to:

sd:/atmosphere/contents/<TITLE_ID>/cheats/<BUILD_ID>.txt
